### Step-by-Step Portfolio Website Structure

If you are creating your portfolio from scratch, follow this order:

## 1. Navigation Bar

**Purpose:** Helps visitors move between sections.

Include:

* Logo/Name (Saravanan M)
* Home
* About
* Skills
* Experience
* Certificates
* Projects
* Contact
* Join Community Button

Example:

```html
<nav>
    <h2>Saravanan M</h2>
    <ul>
        <li><a href="#about">About</a></li>
        <li><a href="#skills">Skills</a></li>
        <li><a href="#projects">Projects</a></li>
        <li><a href="#contact">Contact</a></li>
    </ul>
</nav>
```
